# London Housing Data Analysis

## 📖 Brief Description
This project explores historical housing trends across various boroughs in London. It analyzes average housing prices, number of property sales, and how these factors change over time. The notebook provides visual insights to better understand real estate patterns in the city.

## 🗂️ Data Source
The dataset used for this project is publicly available in kaggle:
https://www.kaggle.com/datasets/justinas/housing-in-london

## 🛠️ How to Run the Project
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/London-Housing-Analysis.git
   cd London-Housing-Analysis
   ```
2. Install the required Python packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook London_Housing.ipynb
   ```
